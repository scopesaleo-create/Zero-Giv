# Product photo slots

Drop product photos at these paths and the site will load them automatically (the SVG fallback fades out):

- `assets/sock-side.png` — hero / interactive sock (side profile showing the sole grip pattern)
- `assets/sock-front.png` — tech section (front view showing ZG monogram)
- `assets/sock-sole.png` — reserved for grip detail view

PNG with transparent background works best. JPG is fine if shot against pure white.
Recommended size: 1200×1200 px, centered subject with ~12% padding.
